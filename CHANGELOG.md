## [1.2] - 2017-05-10
### Added
- 'unsplit' - add files together
- '-v' options for all scripts
- make target for Bourne shell install

## [1.1] - 2017-05-03
### Added
- 'follow' - execute a command at intervals
- '-h' options for all scripts
